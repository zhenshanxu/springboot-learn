# springboot-learn

当前最新 ： 1.0 (项目开发中...)

## 后端框架
- 基础框架 ：Spring Boot
- 持久层框架：Mybatis-plus
- 安全框架：Jwt
- 数据库连接池：阿里巴巴Druid
- 缓存框架：redis
- 其他：Swagger-ui, lombok（简化代码）

## 开发环境
- 语言：Java 8
- IDE（Java）：Eclipse安装lombok插件 或者 Idea
- 依赖管理：Maven
- 数据库：MySQL5.7+ & Oracle 11g
- 缓存：Redis

## 功能简述
- Quartz定时功能
- 二维码生成
